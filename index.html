Mission Erklärung

Deine Aufgabe ist es, das Geheimnis des Hintergrunds zu entdecken und den Code zu knacken.

Auftrag

  1. Hinweise sammeln
  2. Rätsel lösen
  3. Code entschlüsseln
  4. Mission bestätigen
  5. Bericht abgeben

Rätsel 1: Zahlenzauber

Finde die versteckte Zahlenkombination im Hintergrund und entschlüssele sie.

Rätsel 2: Farbcode

Die Farben verraten dir Hinweise. Kombiniere sie richtig, um weiterzukommen.

Rätsel 3: Logikrätsel

Analysiere die Hinweise und löse das logische Puzzle, um den nächsten Schritt zu erhalten.

Rätsel 4: Symbolsuche

Bestimmte Symbole geben dir den Code. Finde alle und setze sie korrekt zusammen.

Rätsel 5: Geheimschrift

Entschlüssle die mysteriöse Nachricht, die dir den finalen Code verrät.
